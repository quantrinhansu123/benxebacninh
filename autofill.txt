● Brainstorm: Auto-fill Form Cấp Phép Điều Độ

  Dựa trên phân tích code hiện tại, đây là các trường có thể tự động điền:

  Các trường ĐÃ tự động (✅)
  ┌────────────────┬──────────────────────┐
  │     Trường     │    Nguồn dữ liệu     │
  ├────────────────┼──────────────────────┤
  │ Số ghế         │ vehicle.seatCapacity │
  ├────────────────┼──────────────────────┤
  │ Số giường      │ vehicle.bedCapacity  │
  ├────────────────┼──────────────────────┤
  │ Đơn vị vận tải │ vehicle.operatorId   │
  ├────────────────┼──────────────────────┤
  │ Ngày xuất bến  │ Mặc định = hôm nay   │
  └────────────────┴──────────────────────┘
  Các trường CẦN bổ sung auto-fill (🔴)
  ┌────────────────────┬───────────────────────────────────────────┐
  │       Trường       │             Đề xuất auto-fill             │
  ├────────────────────┼───────────────────────────────────────────┤
  │ Tuyến vận chuyển   │ Lấy từ chuyến cuối của xe (last dispatch) │
  ├────────────────────┼───────────────────────────────────────────┤
  │ Mã lệnh vận chuyển │ Auto-generate: {RouteCode}-{YYMMDD}-{001} │
  ├────────────────────┼───────────────────────────────────────────┤
  │ Biểu đồ giờ        │ Chọn schedule gần nhất với giờ hiện tại   │
  ├────────────────────┼───────────────────────────────────────────┤
  │ Giờ xuất bến       │ Lấy từ schedule được chọn                 │
  └────────────────────┴───────────────────────────────────────────┘
  Giải pháp đề xuất
